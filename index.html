Resumo sobre HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele define a estrutura do conteúdo na web, utilizando elementos como títulos, parágrafos, links, imagens e outros componentes.

Elementos Básicos do HTML

  • 1. Estrutura Básica
  • 1.1 <html>: Raiz do documento HTML.
  • 1.2 <head>: Contém metadados, título e links para estilos e scripts.
  • 1.3 <body>: Contém o conteúdo visível da página.
  • 2. Texto
  • 2.1 <h1> a <h6>: Títulos de diferentes níveis, onde <h1> é o mais importante e <h6> o menos importante.
  • 2.2 <p>: Define um parágrafo de texto.
  • 3. Destaques e Formatação
  • 3.1 <mark>: Destaca um trecho de texto com uma marcação (geralmente amarela).
  • 3.2 <small>: Exibe o texto em um tamanho menor.
  • 3.3 <i>: Exibe o texto em itálico (ênfase leve).
  • 3.4 <u>: Sublinha o texto.
  • 3.5 <strong>: Dá ênfase forte ao texto, geralmente em negrito.
  • 3.6 <hr>: Insere uma linha horizontal para separar conteúdos.
  • 3.7 <sub>: Insere texto subscrito (abaixo da linha, como em fórmulas químicas).
  • 3.8 <sup>: Insere texto sobrescrito (acima da linha, como em potências).
  • 3.9 <blockquote>: Define uma citação longa, geralmente com recuo.
  • 3.10 <font>: (Obsoleta) Define cor, tipo e tamanho de fonte — não se usa mais em HTML moderno.
  • 3.11 <del>: Indica texto removido ou desatualizado (riscado).
  • 3.12 <ins>: Indica texto inserido (adicionado).
  • 3.13 <abbr>: Define uma abreviação ou sigla, com um atributo title para descrever o significado.
  • 3.14 <pre>: Define texto pré-formatado, preservando espaços e quebras de linha.
  • 3.15 <em>: Indica ênfase leve no texto, geralmente em itálico.
  • 3.16 <cite>: Usado para citar o título de uma obra, como livros ou filmes.
  • 3.17 <code>: Usado para exibir código de programação, geralmente em uma fonte monoespaçada.
  • 3.18 <kbd>: Representa a entrada do usuário, como teclas digitadas no teclado.
  • 3.19 <samp>: Usado para exibir saída de um programa ou exemplo de código.
  • 3.20 <var>: Representa uma variável em um contexto matemático ou de programação.
  • 3.21 <address>: Usado para fornecer informações de contato do autor ou proprietário do documento.
  • 3.22 <time>: Representa uma data ou hora, com atributos para especificar o formato.
  • 3.23 <details>: Cria um widget interativo que pode ser expandido ou recolhido para mostrar ou ocultar informações adicionais.
  • 3.24 <summary>: Usado dentro de <details> para fornecer um resumo ou título que o usuário pode clicar para expandir ou recolher o conteúdo.
  • 3.25 <dialog>: Define uma caixa de diálogo ou janela modal, que pode ser usada para interações com o usuário.
  • 3.26 <progress>: Representa o progresso de uma tarefa, como o carregamento de um arquivo.
  • 3.27 <meter>: Representa uma medição escalar dentro de um intervalo conhecido, como a temperatura ou a velocidade.
  • 3.28 <bdi>: Isola uma parte do texto para evitar que a formatação automática de direção de texto afete o conteúdo.
  • 3.29 <bdo>: Permite alterar a direção do texto, seja da esquerda para a direita (LTR) ou da direita para a esquerda (RTL).
  • 3.30 <wbr>: Sugere um ponto de quebra de linha opcional, permitindo que o navegador quebre a linha nesse ponto se necessário.
  • 3.31 <ruby>: Usado para anotações de pronúncia ou tradução, como em textos japoneses.
  • 3.32 <rt>: Define a anotação de pronúncia ou tradução dentro de um elemento <ruby>.
  • 3.33 <rp>: Usado para fornecer parênteses ao redor de anotações de pronúncia em navegadores que não suportam o elemento <rt>.
  • 3.37 <em>: Exibe texto em itálico, com ênfase semântica.
  • 4. Listas
  • 4.1 <ol>: Cria uma lista ordenada (numerada).
  • 4.2 <ul>: Cria uma lista não ordenada (com marcadores).
  • 4.3 <li>: Define um item de uma lista (<ol> ou <ul>).
  • 4.4 <dl>: Cria uma lista de definição.
  • 4.5 <dt>: Define um termo em uma lista de definição.
  • 4.6 <dd>: Define a descrição ou definição de um termo em uma lista de definição.
  • 4.7 <menu>: Define um menu de comandos ou opções, geralmente usado para menus contextuais.
  • 4.8 <menuitem>: Define um item dentro de um menu, como uma opção que o usuário pode selecionar.
  • 4.9 <dir>: (Obsoleta) Usada para criar uma lista de diretórios, não é mais recomendada.
  • 4.10 <li>: Define um item de lista, usado dentro de <ol>, <ul> ou <menu>.
  • 4.11 <dl>: Cria uma lista de definição, onde <dt> define o termo e <dd> define a descrição.
  • 4.12 <dt>: Define um termo em uma lista de definição, usado dentro de <dl>.
  • 4.13 <dd>: Define a descrição ou definição de um termo em uma lista de definição, usado dentro de <dl>.
  • 4.14 <menu>: Define um menu de comandos ou opções, geralmente usado para menus contextuais.
  • 4.15 <menuitem>: Define um item dentro de um menu, como uma opção que o usuário pode selecionar.
  • 4.16 <dir>: (Obsoleta) Usada para criar uma lista de diretórios, não é mais recomendada.
  • 4.17 <li>: Define um item de lista, usado dentro de <ol>, <ul> ou <menu>.
  • 5. Links e Imagens
  • 5.1 <a>: Cria um link para outra página ou recurso.
  • 5.2 <href>: Define um link para outra página ou recurso.
  • 5.3 <img>: Insere uma imagem na página, com atributos src (fonte) e alt (texto alternativo).
  • 5.4 <figure>: Agrupa conteúdo relacionado a uma imagem, como legendas.
  • 5.5 <figcaption>: Define uma legenda para uma imagem dentro de <figure>.
  • 5.6 <map>: Define um mapa de imagem, permitindo áreas clicáveis dentro de uma imagem.
  • 5.7 <area>: Define uma área clicável dentro de um mapa de imagem, com atributos para especificar a forma e o link associado.
  • 5.8 <object>: Insere um objeto multimídia, como um vídeo ou animação, com atributos para especificar o tipo e a fonte do conteúdo.
  • 5.9 <embed>: Insere um recurso externo, como um vídeo ou áudio, com atributos para especificar a fonte e o tipo do conteúdo.
  • 5.10 <iframe>: Insere um quadro inline, permitindo incorporar outra página HTML dentro da página atual.
  • 5.11 <canvas>: Cria uma área de desenho em 2D, onde é possível renderizar gráficos e animações usando JavaScript.
  • 5.12 <svg>: Define um gráfico vetorial escalável, permitindo criar imagens e formas complexas usando XML.
  • 5.13 <video>: Insere um vídeo na página, com atributos para especificar a fonte, controles e opções de reprodução.
  • 5.14 <audio>: Insere um áudio na página, com atributos para especificar a fonte, controles e opções de reprodução.
  • 5.15 <source>: Define uma fonte alternativa para um recurso multimídia, como um vídeo ou áudio, permitindo especificar diferentes formatos e qualidades.
  • 5.16 <track>: Define uma faixa de texto para um vídeo, como legendas ou descrições, com atributos para especificar o tipo e a linguagem da faixa.
  • 5.17 <map>: Define um mapa de imagem, permitindo áreas clicáveis dentro de uma imagem.
  • 5.18 <area>: Define uma área clicável dentro de um mapa de imagem, com atributos para especificar a forma e o link associado.
  • 5.19 <object>: Insere um objeto multimídia, como um vídeo ou animação, com atributos para especificar o tipo e a fonte do conteúdo.
  • 5.20 <embed>: Insere um recurso externo, como um vídeo ou áudio, com atributos para especificar a fonte e o tipo do conteúdo.
  • 6. Tabelas
  • 6.1 <table>: Define uma tabela.
  • 6.2 <tr>: Define uma linha de tabela.
  • 6.3 <th>: Define um cabeçalho de tabela.
  • 6.4 <td>: Define uma célula de dados na tabela.
  • 6.5 <thead>: Agrupa o cabeçalho de uma tabela, contendo linhas de cabeçalho.
  • 6.6 <tbody>: Agrupa o corpo de uma tabela, contendo linhas de dados.
  • 6.7 <tfoot>: Agrupa o rodapé de uma tabela, contendo linhas de resumo ou totais.
  • 6.8 <caption>: Define uma legenda para uma tabela, descrevendo seu conteúdo.
  • 6.9 <col>: Define uma coluna em uma tabela, permitindo aplicar estilos ou atributos específicos a uma coluna inteira.
  • 6.10 <colgroup>: Agrupa um conjunto de colunas em uma tabela, permitindo aplicar estilos ou atributos a várias colunas de uma só vez.
  • 7. Formulários
  • 7.1 <form>: Define um formulário para entrada de dados.
  • 7.2 <input>: Cria um campo de entrada de dados, com diferentes tipos (texto, senha, etc.).
  • 7.3 <label>: Define um rótulo para um campo de entrada.
  • 7.4 <button>: Cria um botão clicável.
  • 7.5 <select>: Cria uma lista suspensa para seleção de opções.

1. Estrutura Básica DO HTML

Ir para seção

Resumo: Todo documento HTML começa com a declaração do tipo de documento, seguido pelas tags <html>, <head> e <body>.

Exemplo

  1. <!DOCTYPE html>
  2. <html lang="pt-BR">
  3. <head>
  4. <title>Título da Página</title>
  5. </head>
  6. <body>
  7. Conteúdo da página aqui
  8. </body>
  9. </html>

Conclusão

O HTML é a base estrutural de qualquer página web. Ele permite organizar e formatar o conteúdo de forma clara, acessível e eficiente. Compreender o uso correto de suas tags e atributos é fundamental para criar páginas bem construídas, funcionais e compatíveis com diversos dispositivos e navegadores.
